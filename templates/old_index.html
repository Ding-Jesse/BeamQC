⚠️注意事項

  1. CAD版本須為2016以前之版本
  2. 在title後面加上括號(ex. 地下五層結構平面圖(B5F), 五層至六層結構平面圖(5F-6F))

    This is good.(有括號)

    This is not good. (沒有括號)

  3. 圖框的圖層要一致

  4. 顯示樓層的字串中，'2-5F', '3F~6F', '3F,5F,7F' 等等都是可接受的，但盡量避免中文字元(可能會變成西班牙文亂碼)

    This is not good. (頓號變成'¡B'了)


  5. 在一個圖框中，盡量只放一張圖，如果要放兩張以上的圖，請將黃色標題放在圖的下方，比較不容易出錯。

    This is good.(黃色標題都在圖的下方)

    This is not good. (黃色標題都在圖的右方，梁跟樓層容易對錯)
